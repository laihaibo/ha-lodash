# hadash
The `Hadash` library exported as `Node.js` modules.

## Installation
Using npm:
```
npm i --save hadash
```
In Node.js
```
// Load the full build.
const ha = require('hadash');

// Load methods. 
const {chunk} = require('hadash');
```
